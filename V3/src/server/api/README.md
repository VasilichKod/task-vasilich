# API Layer

Сюда следующим шагом кладём:
- `groups`
- `projects`
- `weekly_tasks`
- `backlog_tasks`
- `recurring_tasks`
- `achievements`
- `settings`

Рекомендуемый порядок:
1. `GET /workspace/bootstrap`
2. CRUD для групп и проектов
3. CRUD для недельных задач
4. CRUD для backlog задач
5. CRUD для достижений
6. профиль и настройки
